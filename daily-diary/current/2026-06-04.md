### What we did

Two Figma-alignment fixes for carousel nav arrows:

**1. Wrong icon shape**  
Code had pure chevron SVGs (`M14 5l-6 6 6 6` in 22×22 viewBox). Figma uses `icon--ui-arrow_left--line` / `icon--ui-arrow_right--line` — arrow-with-shaft. Fixed by swapping paths + updating viewBox to `0 0 18 18`. Cross-referenced correct paths from `event-list` block which already had them.

**2. Disabled state color**  
`navArrow` disabled selector was missing `color` entirely — icon stayed full black when disabled. Figma variable `control/tertiary/disabled/text: #00000033` = `rgba(0,0,0,0.2)`. SVG uses `stroke="currentColor"` so setting `color` on button propagates to icon. Also changed `borderColor: transparent` → `borderColor: rgba(221,223,216,0)` to match `control/tertiary/disabled/border: #dddfd800`.

Both changes in:
- `projects/xsolla-com/featured-articles/src/block.tsx` (SVG paths + viewBox)
- `projects/xsolla-com/featured-articles/src/block.css.ts` (disabled selector)
