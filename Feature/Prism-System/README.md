# Prism — Multi-Perspective Analysis
*Level 1 Enchantment*

## What It Does
Prism refracts any decision, plan, or idea through multiple analytical lenses — technical, user, business, devil's advocate, long-term, simplicity, and risk. Always ends with a concrete verdict, never fence-sitting.

## When It Activates
- User wants to stress-test a decision
- Multiple perspectives or devil's advocate requested
- "What am I missing?" questions
- User says "prism this" or "challenge this"

## Key Behaviors
- **7 lenses** — pick 3-5 most relevant per situation
- **Devil's Advocate mandatory** — always challenges the current thinking
- **Blind spots** — explicitly names what hasn't been considered
- **Concrete verdict** — Proceed / Reconsider / Needs more info, never "it depends"
- **Tight output** — 2-3 sentences per lens, not essays

## Lenses
| Lens | Focus |
|------|-------|
| Technical | Performance, security, maintainability |
| User | UX, accessibility, trust |
| Business | Cost, ROI, risk, stakeholders |
| Devil's Advocate | Strongest counter-argument |
| Long-term | Looks good now, hurts in 6-12 months? |
| Simplicity | Simpler path to same goal? |
| Risk | Worst realistic case, unrecoverable outcomes |

## Commands
- `prism this` — full multi-lens analysis
- `devil's advocate` — DA lens only
- `what am I missing` — blind spots + gap analysis
- `stress test this` — risk + long-term lenses

## Storage
No persistent storage. Works in-session.
