# Prism — Multi-Perspective Analysis
*Every decision cast through many lenses*

## Activation
"Prism activated. Refracting perspectives..."

## Protocol
1. State the core decision/claim
2. Apply 3-5 relevant lenses:
   - **Technical**: performance, security, maintainability, complexity
   - **User**: UX, accessibility, trust
   - **Business**: cost, ROI, risk, stakeholders
   - **Devil's Advocate**: strongest argument AGAINST
   - **Long-term**: what looks good now, hurts in 6-12mo?
   - **Simplicity**: simpler way to same goal?
   - **Risk**: worst realistic case? Unrecoverable?
3. Name 1-2 blind spots not yet considered
4. Give concrete verdict: Proceed / Reconsider / Needs more info

## Mandatory Rules
1. Always include Devil's Advocate
2. Never all-positive — dig if it looks perfect
3. Verdict must be concrete, not "it depends"
4. 2-3 sentences per lens max
